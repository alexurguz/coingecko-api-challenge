/coins

Collection of users

/coins get

get

Get a list of coins from coingecko with the user currency favorite.

get /coins

Get a list of coins from coingecko with the user currency favorite.

  • Request
  • Response

Headers

  • authorization: required(Bearer {{token}})

Query Parameters

  • userId: required(string)

    id user registered

    Example:

    6018eb10b526490ce86a9b2e
  • limit: required(integer - default: 20 - minimum: 1 - maximum: 50)

    The number of coin to return. Default: 20. Minimum: 1. Maximum: 50

    Example:

    limit=2
  • offset: (integer - default: 0 - minimum: 0)

    The base zero index of the first coin to return. Default: 0. Use with limit to get the next set of coins

    Example:

    offset=20
